## 🎨 Design System

### Colors
- **Primary**: Healthcare Blue (#0066cc)
- **Success**: Soft Green (#4caf50)
- **Warning**: Amber (#ffc107)
- **Error**: Red (#f44336)

### Typography
- Font Family: Inter
- Heading: Semibold
- Body: Regular

### Spacing
- 8px base spacing system
- Consistent padding and margins

### Components
- Buttons (Primary, Secondary, Destructive)
- Input Fields
- Cards
- Status Badges
- Modals
- Tables
- Navigation
